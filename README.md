# ASCII Art converter

This project is a web app that converts images to ASCII Art

## How to use it

Open the app url, and then upload any image and select width size, then click on convert, you'll get a ASCII image.

## Features

- Adjustable output width
- Copy the art
- Download the art as .txt file

## Coding languages

- Front end: HTML, CSS, vanilla JavaScript
- Backend: Python (flask)
- Deployed on Vercel

## How to access the app
- link: https://ascii-art-navy.vercel.app
